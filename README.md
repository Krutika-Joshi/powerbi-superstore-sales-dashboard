# 📊 Super Store Sales Dashboard | Power BI

## 📌 Project Overview

This project is an end-to-end **Power BI Sales Dashboard** developed by implementing a guided project to strengthen my **Business Intelligence**, **Data Visualization**, and **Dashboard Development** skills.

The dashboard provides interactive insights into sales performance, profit, customer segments, shipping methods, regional performance, and sales forecasting using the Super Store Sales dataset.

## 📈 Dashboard Features

- Sales, Profit & Quantity KPIs
- Average Delivery Time
- Sales by Region
- Sales by Segment
- Sales by Category & Sub-Category
- Sales by Payment Mode
- Sales by Ship Mode
- Monthly Sales & Profit Trends
- Interactive State Map
- Region Slicer
- 15-Day Sales Forecast
